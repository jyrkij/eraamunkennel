Tietoja minusta

Nimeni on Jyrki Lilja. Opiskelen tietotekniikkaa Rovaniemen ammattikorkeakoulussa. Valmistuminen häämöttää näillä näkymin vuoden 2012 joulukuussa. Valo- ja videokuvaus ovat lähellä sydäntäni, siksipä näillä sivuilla tulette näkemään paljon kuvia.

Ympäristöistä OS X ja Windows ovat tutuimmat. Ohjelmointikielistä sujuvat mm. C, Objective-C, C++, C#, Java, Ruby, Python sekä WWW-sivustoihin liittyvät PHP, HTML5 ja CSS3.
